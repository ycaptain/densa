# Chronos skeletons (dated, dense, zoomable timelines)

Requires the Chronos Timeline plugin. Use for awareness-progress
axes, route signal timelines, interaction histories — anything with
real dates where density matters.

## Basic events + periods

````markdown
```chronos
- [2026-03-12] 首次命名该机制
- [2026-04-02~2026-04-30] 阶段:塌方期 #orange
- [2026-05-27] 镜像场(丁)
- [2026-05-28] 镜像场(苑)
```
````

## Grouped lanes (e.g. one lane per route / per person)

````markdown
```chronos
- [2026-04-10] {Route A} 信号:JD 更新
- [2026-05-02] {Route A} 决策:暂缓
- [2026-05-15] {Route B} 内推线索
```
````
